# Pure components and storybook

<DocCard>
  <Text>Pure components (all components under <code>webapp/src/component</code> directory) can't use or modify global
    variables (<code>import history from '../history'</code> <code>useDispatch</code> <code>useSelector</code>), can't
    make network request (The use of <code>GoogleLoginButton</code> or <code>GoogleMap</code> components), and can't use
    any components under <code>webapp/src/container</code> directory.</Text>
</DocCard>
